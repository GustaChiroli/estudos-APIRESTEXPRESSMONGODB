Node 18.16.0
Npm 9.5.1

Para rodar a api é necessário.
 - Clonar do github.
 - Rodar no terminal o comando cd APIRESTEXPRESSMONGODB (ou o diretório que vc o colocou).
 - Ter instalado ou instalar o node da versão correta.
 - Rodar o comando npm install.
 - Configurar o mongoDB para o projeto.

Lembrete: informações como a string de conexão do banco estão guardadas no .env que não é enviado para o github, tornando assim necessário para cada clone uma configuração do mongoDB.
